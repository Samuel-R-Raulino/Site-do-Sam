Liderei um time de programação para a criação de um site não disponivel a público ainda chamado OraGame. Além de liderar fiz 70% da back-end do site, incluindo a comunicação com banco de dados, o cadastro de contas, os logins, as abas de games, e o sistema de fliperama. O site é basicamente uma versão Frutiger Aero da Steam, os jogos colocados são de outras empresas pois não tinhamos jogos prontos e não tinhamos a intenção de jogar a OraGame no mercado.
Utilizei o Python Flask para fazer o servidor do site e sua comunicação com o banco de dados. Usei o Jinja2 para poder usar a lógica de programação dentro da linguagem de modelagem do HTML. Usei também o JavaScript para a criação de um sistema de fliperama que roda games de MS-DOS, através da API JS-DOS.
